## Acceptance Criteria

```
GIVEN I need a new, random number
WHEN I click the button to generate a number
THEN I am presented with a series of prompts for number criteria
WHEN prompted for number criteria
THEN I select which criteria to include in the number
WHEN prompted for the length of the number
THEN I choose a length of at least 3 and no more than 10
WHEN prompted for character types to include in the number
THEN I choose either even or odd, and negative or positive
WHEN I answer each prompt
THEN my input should be validated
WHEN all prompts are answered
THEN a number is generated that matches the selected criteria
WHEN the number is generated
THEN the number is either displayed in an alert or written to the page
```
